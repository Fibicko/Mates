## Možnosti testování

Upravit konstantu zapnoutTestovani na hodnotu TRUE. Poté možné pracovat s údaji jako jsou: den, měsíc, rok, hodina.

    const zapnoutTestovani = false; // nastavit na true pro testování

    const testMesic = 10;
    const testRok = 2022;
    const testHodina = 14;
